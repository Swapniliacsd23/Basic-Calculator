#  Basic Calculator – Python Project

This is a beginner-friendly calculator project implemented in Python.  
It includes:

-  A Command-Line Calculator
-  A GUI Calculator using Tkinter
-  Unit Tests using Python’s `unittest` module

---

##  Project Directory

Basic Calculator/

```bash
